# Havoc Profile Generator by 0xv1n

Purpose: Simple GUI tool to generate a Havoc C2 profile for Demon.

![](/assets/mainwindow.png)

## Usage

1. `go run main.go`
2. fill in parameters as desired
3. click save
4. profile is saved as `profile.yaotl`.
